博客改版记录

3 月 25 日，2026 博客 更新于 3 月 25 日，2026 约 1 分钟阅读 497 字

这次把站点从默认主题收回到一套更轻、更白、更适合长期写作的内容主题里。首页、文章页、图库、归档和评论系统现在已经统一起来，后续更新只需要专注在内容本身。

背景

原先站点还是 Hexo 默认结构和默认内容，页面虽然能用，但信息层级、视觉质感和后续维护成本都不太对。尤其是默认的侧栏、配色和占位文案，会让整个博客看起来像刚初始化完成，还没真正开始写作。

这次改版的目标很明确：

  • 视觉上改成简约白底分格
  • 结构上更适合内容型博客长期积累
  • 功能上补齐评论、图库、归档和 RSS
  • 写作上让后续更新尽量只围绕 Markdown 本身

过程

这一版主题围绕四个核心页面去收结构：首页负责精选和最近更新，文章页负责阅读体验，图库页负责集中整理图片，关于页负责替代原来过度泛化的个人介绍模板。

同时把文章 scaffold 也整理好了，后续每次新建文章时都会带上这些常用字段：

  • title
  • subtitle
  • description
  • cover
  • series
  • categories
  • tags
  • featured
  • comments

这样更新的时候不用每次临时想 front matter 应该怎么写，直接填内容就行。

评论系统现在接的是 utterances，也就是把评论落在 GitHub Issues 里。对个人博客来说这套方案很轻，不需要自己维护服务端。

结论

现在这套博客已经不是“演示站”状态了，而是可以直接持续更新的内容站。接下来你只需要把封面、图片和文章不断替换成自己的真实内容，整站就会越来越完整。

后续我会继续沿着这个方向迭代：让文章页更稳、内容更真、页面更少但更有用。
